AutoSorter

Bem vindo a aplicação AutoSorter, uma aplicação de organização de listas que usa algoritmos de ordenação! Esse projeto tem como objetivo mostrar como funciona um sistema de organização de listas, desde organizar contatos, notas, valores númericos, até sistemas complexos que precisam ordenar e buscar muita quantidades de dados. 

Nessa aplicação usamos três tipos de algoritmos de ordenação para ordenar essas listas, são eles : Bubble sort ; Quick sort e Binary search; 

• Bubble sort (Ordenação por bolha) : O Bubble sort é um algoritmo que ordena uma lista comparando elementos próximos/sequintes e trocando-os para o lugar correto quando eles estiverem localizado errado; 

• Quick sort (odenação rápida) : O Quick sort é um algortimo que ordena por meio da divisão, ele escolhe um Elemento e os elementos menores vão para um lado(esquerdo) e os elementos maiores vão para o outro lado(direito); 

• Binary search (Busca binária) : é um método rápido para encontrar um valor em um array ordenado, dividindo a busca ao meio a cada passo. Ela compara o valor do meio com o que você procura e, dependendo se é maior ou menor, descarta metade dos elementos, repetindo o processo até encontrar ou acabar os dados. Por eliminar 50% das opções a cada comparação, ela é muito eficiente e funciona em O(log n), sendo ideal para listas grandes, desde que estejam ordenadas.

    Objetivos dessa aplicação :

 Demostrar de forma didática e prática o funcionamento desses algoritmos de ordenação e destacar : 

 • Vantagens e Desvantagens do uso de cada algoritmo; 

 • Tempo de execução de cada algoritmo; 

 • A complexidade dos algortimos;

 • Suas aplicações no cotidiano; 

    Funcionalidades :

 • Comparação entre os algoritimos Bubble sort, Quick sort e Binary search;

 • Demostração na prática do tempo de execução de cada algoritmo;

 • Visualização de como cada algoritmo funciona;
